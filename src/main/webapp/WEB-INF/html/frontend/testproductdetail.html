关于悦享校园

悦享校园旨在基于大学生生活现状需求，打造可以提供综合的校园服务于一体的校园商铺平台。以“便捷、快速、优质”为目标，使大学生能愉悦的享受校园生活。

悦享校园是基于SSM框架开发的在线o2o商铺，前端页面使用SUI Mobile的UI库，通过ajax与后端数据交互，实现前后端分离。因时间仓促，项目难免有不足之处，项目仍在不断优化中。若在体验中有更好的建议与看法，欢迎反馈。
